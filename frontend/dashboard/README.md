# LeagueFindrDashboard

This is the frontend for the LeagueFindrDashboard.

## Getting Started

First, run the development server:

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- User Authentication
- User League Management
- User League Finder

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- ESLint
- Prettier
